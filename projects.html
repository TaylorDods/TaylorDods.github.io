Large Language Model File Integration

This project aims to develop a versatile desktop application that allows uploading and organizing files of various formats to a MongoDB database. Utilizing a hybrid approach, the application stores each file alongside a vector embedding of its content, enhancing data retrieval through large language models. Currently, the back-end functions are implemented, and work is underway to develop the vector store for search functionality.

Shell Eco-Marathon: Chassis

Our team designed and implemented a lightweight, energy-efficient chassis for the Shell Eco-Marathon competition, utilizing technologies such as SolidWorks, ANSYS, and python.
